# Part 2: 台本テンプレート

## 基本構成

```
OP（フック） → 起（理不尽な現実の提示） → 承（理不尽の拡大） 
→ 転①（小逆転） → 転②（危機） → 転③（大逆転） → 結（平和な日常）
```

## セクション1：OP（フック）

### 目的
- 視聴者の興味を一瞬で掴む
- 「何が起きるんだ？」と続きを見たくさせる

### 含める要素
- 悪役の印象的なセリフ（怒鳴り声、暴言など）
- 主人公が理不尽な状況にいることを示唆
- 逆転の予告（「だが、状況は一変した」など）
- 主人公の正体を匂わせる（名刺、肩書きなど）

### 例文・参考
```
「お前みたいな使えない奴は、今すぐクビだ！」
部長の怒鳴り声が、オフィスに響いた。
（中略）
俺の名刺には、こう書いてあった。
「本社 内部監査室 室長 村田健太」
これは、パワハラ上司と、潜入調査で正体を隠していた男の物語だ。
```

## セクション2：起（理不尽な現実の提示）

### 目的
- 状況設定を説明する
- 悪役のムカつく行動を具体的に描写する
- 視聴者の怒りを煽る

### 含める要素
- 主人公の本当の立場・目的の説明
- 悪役との出会いのシーン
- 悪役の具体的な暴言・行動（複数）
- 悪役自身のダメな行動との対比
- 主人公が証拠を集めている描写

### 例文・参考
```
「お前、どこの大学だ」
「地方の私立です」
「はあ？俺は早稲田だぞ。そんな三流大学の奴が、よくうちに入れたな」
```

## セクション3：承（理不尽の拡大）

### 目的
- 悪役の行動をエスカレートさせる
- 視聴者のフラストレーションを最大化する
- 逆転への期待を高める

### 含める要素
- 悪役の暴言・行動がさらにひどくなる
- 周囲が見て見ぬふりをする描写
- 主人公が「クビ」など最悪の宣告を受ける
- 主人公の内心の勝利確信（伏線）
- 五感を使った情景描写

### 例文・参考
```
佐藤が俺の顔を覗き込む。コーヒーとタバコの混じった息が、鼻につく。
周りの社員が、こちらを見ている。目が合うと、すぐに逸らされる。
誰も助けない。皆、佐藤を恐れている。
```

## セクション4：転①（小逆転）

### 目的
- 主人公の正体を明かす
- 悪役に最初の打撃を与える
- 視聴者にカタルシスの予感を与える

### 含める要素
- 主人公が正体を明かすシーン
- 悪役の動揺・顔面蒼白の描写
- 証拠の提示（録音、メール、証言など）
- 具体的な数字（〇通、〇件など）
- 悪役の膝が崩れる／震える描写

### 例文・参考
```
俺は、名刺を差し出した。
「本社内部監査室、室長の村田です」
「潜入調査をさせていただきました」
佐藤の顔から、血の気が引いた。
```

## セクション5：転②（危機）

### 目的

- 悪役が再び余裕を取り戻す描写
- 主人公の絶望・焦りの描写
- 周囲の諦めムード

### 例文・参考
```
「やっぱり、証拠不十分だよなあ？」
佐藤が、田中の肩を叩いた。
「残念だったな、村田くん」
「お前の負けだ」

- 懲戒解雇・退職金不支給などの宣告
- 希望の光が戻ってくる描写

### 例文・参考
```
「佐藤、本日付で懲戒解雇。退職金は全額不支給」
佐藤の膝が、崩れ落ちた。
「許してくれ…何でもする…」

「あなたのせいで退職した人たちに、その言葉を言ってください」
```

# Part 3: 台本執筆ルール

## 1. 文章の基本ルール

### 避けるべき表現
- 説明的な地の文（感情は描写で見せる）
- 長い複文（一文に情報を詰め込みすぎない）
- 「〜だと思った」などの曖昧な心理描写

### 推奨する表現
- 短文の連続でテンポを作る
- 体言止めで余韻を残す（「典型的な『仕事しない上司』だった。」）
- セリフと地の文を交互に配置
- 五感を使った情景描写（視覚、聴覚、嗅覚など）

### 文の長さ・リズム
- 一文は20〜40文字程度を基本とする
- 重要なシーンは短文（10文字以下）で畳みかける
- 「＊＊＊」で場面転換を明示

---

## 2. ジャンル特有のルール

### このジャンルで重要なこと
- 悪役の描写を具体的かつ「ムカつく」ものにする
- 悪役のセリフは見下し・侮辱・自慢を入れる
- 悪役自身のダメな行動を対比として描く（サボり、横取りなど）
- 周囲が見て見ぬふりをする「孤立感」を演出
- 逆転の証拠は具体的な数字で示す（〇通、〇件、〇名分）

### 絶対にやってはいけないこと
- 悪役を中途半端に描く（徹底的にムカつかせる）
- ご都合主義の展開（逆転には論理的な根拠を用意）
- 主人公が感情的になりすぎる（冷静に淡々と進める）
- オチを最初から見せすぎる（フックでは匂わせる程度に）

---

## 3. 効果的な表現・ワード

### 要所で使うと効果的な表現
| 表現 | 使いどころ |
|------|------------|
| 「顔から血の気が引いた」 | 悪役が追い詰められた瞬間 |
| 「膝が崩れ落ちた」 | 悪役の敗北シーン |
| 「フロアが静まり返った」 | 緊張感を高める場面転換 |
| 「その時」 | 逆転の起点 |
| 「だが」「しかし」 | 展開の転換 |

### 避けるべきワード・表現
| 表現 | 理由 |
|------|------|
| 「すごく」「とても」 | 抽象的で映像が浮かばない |
| 「〜と思った」 | 説明的。行動や表情で見せる |
| 「実は〜だったのだ」 | 説明口調になりがち |

---

## 4. 視聴者との関係性

### どのくらいの頻度で意識するか
- ストーリー中は視聴者への直接呼びかけは控える
- 最後の「結」パートでのみ視聴者に語りかける

### 関係性を作るための手法
- 主人公の内心を短く挟む（「俺は、心の中で呟いた」）
- 悪役への怒りを視聴者と共有する描写
- 結末で「〇〇に苦しんでいる人へ」と励ます
- 教訓を押し付けず、さりげなく伝える

---

## 5. その他のルール

### セリフの書き方
- セリフは「」で囲む
- セリフの前後に話者の動作や表情を入れる
- 悪役のセリフは短く、インパクト重視
- 「w」は悪役のセリフやサムネ・タイトルで使用可

### 場面転換のルール
- 「＊＊＊」で大きな場面転換
- 「---」でセクションの区切り
- 時間経過は「一週間前。」「金曜日の夕方。」など冒頭に明示

### 2. 感情の起伏・感情曲線

**この台本の感情の流れ：**

```
