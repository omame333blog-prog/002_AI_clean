# リスト報告テンプレート（スプレッドシート用）

報告用のテンプレートをシートで入力しやすくし、**報告文を1セルで完成**させる構成案です。

---

## まずやること（3ステップ）

1. **チャレロン用スプレッドシート**を開き、**新しいシート「リスト報告」**を1枚追加する。
2. **リスト報告_入力テンプレート.csv** を **ファイル → インポート** でこのスプレッドシートに読み込む（または、下の「1. シートのレイアウト案」を見て A列に項目名、B列に入力欄を手で作る）。
3. **このmdの「3. 報告文を1セルで作る式」** をコピーし、リスト報告シートの **B14** に貼り付ける。

あとは **B2〜B13** に数値を入れる（手入力）か、**「5. 既存シートから参照する」** の式を B2〜B11 に貼ると、オプチャリスト数・目標数値の「出てる数字」が自動で入り、**B14 に報告文ができる**。そのセルをコピーしてチャットやメールに貼れば報告完了。**広告とオーガニックは合計1つの目標**なので、目標まで・日割は1セットだけです。

---

## 1. シートのレイアウト案（広告／オーガニック＝合計1つの目標）

今回のように **広告とオーガニックで「合わせた1つの目標」** にする場合の配置です。内訳（広告○人・オーガニック○人）は表示しつつ、目標まで・日割目標・日割実績は **1セットだけ** にします。

| 列A（項目名） | 列B（入力または参照） | 備考 |
|---------------|------------------------|------|
| **【入力エリア】** | | |
| 広告_人数 | 17 | **全累計**。オプチャリスト数シート **N列1行目** を参照 |
| 広告_前日比 | +17 | 任意（累計表示の場合は空欄でも可） |
| オーガニック_人数 | 29 | **全累計**。オプチャリスト数シート **O列1行目** を参照 |
| オーガニック_前日比 | +29 | 任意（累計表示の場合は空欄でも可） |
| 広告／オーガニック_目標まで | 3928 | 数値（**合計**の目標まで） |
| 広告／オーガニック_日割目標 | 127 | 数値（合計の日割目標） |

## 2. 行番号で揃えた配置例（B列）

- **B2** 広告_本日人数  
- **B3** 広告_前日比  
- **B4** オーガニック_本日人数  
- **B5** オーガニック_前日比  
- **B6** 広告／オーガニック_目標まで（**合計で1つ**）  
- **B7** 広告／オーガニック_日割目標（**合計で1つ**）  
- **B8** 広告／オーガニック_日割実績（**合計**＝B2+B4 の式でOK）  
- **B9** メルマガ合計  
- **B10** 特設LINE合計  
- **B11** オプチャ合計

A列の1行目から順に、上の「列A（項目名）」を入れておく。

## 3. 報告文を1セルで作る式（B14 に貼る想定・合計目標用）

B2〜B13 に入力がある前提で、次の式を **報告文用のセル（B14）** に入れます。  
出力例の形に合わせています。

```
="◆広告："&B2&"人（前日比"&B3&"人）"&CHAR(10)&
"◆オーガニック："&B4&"人（前日比"&B5&"人）"&CHAR(10)&
"　目標まで"&B6&"人"&CHAR(10)&
"　日割り目標未達成：目標"&ROUND(B7,0)&"人/実績"&B8&"人"&CHAR(10)&
CHAR(10)&
"◆登録者数"&CHAR(10)&
"メルマガ合計："&B9&"人"&CHAR(10)&
"特設LINE合計："&B10&"人(メルマガとの差異"&(B10-B9)&"/"&IF(B9=0,"-",TEXT(B10/B9,"0.0%"))&")"&CHAR(10)&
"オプチャ合計："&B11&"人(特設LINEとの差異"&(B11-B10)&"/"&IF(B10=0,"-",TEXT(B11/B10,"0.0%"))&")"&CHAR(10)&
CHAR(10)&
"▼目標管理シート"&CHAR(10)&
B12&CHAR(10)&
CHAR(10)&
"▼リスト数シート"&CHAR(10)&
B13
```

**出力例**
```
◆広告：17人（前日比+17人）
◆オーガニック：29人（前日比+29人）
　目標まで450人
　日割り目標未達成：目標14人/実績46人

◆登録者数
メルマガ合計：78人
特設LINE合計：76人(メルマガとの差異-2/97.4%)
オプチャ合計：67人(特設LINEとの差異-9/88.2%)

▼目標管理シート
URL

▼リスト数シート
URL
```

※ **広告（B2）・オーガニック（B4）**はオプチャリスト数の **N1・O1** を参照すると**全累計**になります。前日比（B3・B5）は累計表示の場合は空欄でOK。  
※ **メルマガ合計（B9）・特設LINE合計（B10）・オプチャ合計（B11）**は、**データ行（B1）で指定した「その日」の人数**です。B1を前日の行にしている場合は「昨日の人数」になります。  
※ 実績（B8）は**その日の**広告+オーガニック合計です。B7（日割目標）は **ROUND(B7,0)** で整数表示にしています。

- `CHAR(10)` で改行しています。
- メルマガとの差異は「特設LINE−メルマガ」、割合は特設LINE/メルマガです。必要なら「メルマガとの差異」の計算を変えてください。
- 特設LINEとの差異は「オプチャ−特設LINE」、割合はオプチャ/特設LINEです。

---

## 4. 使い方

1. 新しいシート「リスト報告」を作る。
2. A列に項目名、B列に対応する行に数値・URLを入力（上記の行番号例のとおり）。
3. 報告文のセル（**B14**）に上の式を貼る。
4. 報告文のセルをコピーして、チャットワークやメールに貼り付けて報告する。

---

## 5. 既存シートから「出てるデータ」を参照する（手動入力ほぼ不要）

オプチャリスト数・目標数値シートにすでにある数値を使うようにすると、**手動入力はURLと「データ行」だけ**で済みます。

### 前提：B1 に「データ行」を入れる

**B1 には次のどちらかを入れます。**

- **手動**：オプチャリスト数の **何行目** を使うか（2/6＝**3**、2/7＝**4** …）を数字で入力。
- **自動（今日の行）**：下の式を B1 に貼る。

#### B1 に貼る式（報告対象の行を自動取得＝前日）

**前日の報告を翌朝にする**（例：6日のデータを7日の朝に報告）なら、**前日の行**を取る式にします。

オプチャリスト数の日付が **B3 から下** に並んでいるとき：

```
=IFNA(MATCH(TODAY()-1,'オプチャリスト数'!B3:B50,0)+2, 2+COUNTA('オプチャリスト数'!B3:B))
```

- **TODAY()-1** で「前日」の日付。7日に開けば6日の行、8日に開けば7日の行になる。
- **MATCH(..., 0)+2** で、その日付の行番号（3, 4, 5…）を出している。
- **IFNA(..., 2+COUNTA(...))** は、前日の日付が一覧にないときは、**日付が入っている最後の行**を使うため。

オプチャリスト数のシート名が **「オプチャリスト数」でない** 場合は、式の中の `'オプチャリスト数'` を実際のシート名に書き換えてください。

### B2〜B11 に貼る「参照の式」（合計目標用・コピペ用）

以下の式を **該当セルに1つずつコピーして貼り付けて**ください。シート名が「オプチャリスト数」「目標数値」でない場合は、式内のシート名を書き換えてください。

- **前日比**（B3, B5）：増減を +17 / -5 のように表示。データ行が3（1日目）のときは空白。
- **B6, B7**：目標数値シートの「**広告+オーガニック合計**」用のセルを参照します。該当セルが無い場合は手入力。
- **B12・B13**（URL）は **手入力** のままにしてください。

---

**B2**（広告・**全累計**＝オプチャリスト数 **N列1行目**）

```
='オプチャリスト数'!N1
```

※ **N列は必ず1行目**を参照します。`=INDIRECT("'オプチャリスト数'!N"&$B$1)` だと「その日の行」になってしまうので使わず、上記 **N1** にしてください。シート名が「オプチャリスト数」でない場合は書き換えてください。

**B3**（広告_前日比）※N1で累計表示する場合は**空欄**または「-」でOK。前日比も出したい場合は、その日の増加分を計算する式を別途入れてください。

**B4**（オーガニック・**全累計**＝オプチャリスト数 **O列1行目**）

```
='オプチャリスト数'!O1
```

※ **オーガニックの全人数（累計）**を表示します。

**B5**（オーガニック_前日比）※O1で累計表示する場合は**空欄**または「-」でOK。

**B6**（広告／オーガニック_目標まで・**合計1つ**）※目標数値の「リスト（オプチャ）」など合計目標のセルに合わせて変更

```
='目標数値'!G4-INDIRECT("'オプチャリスト数'!M"&$B$1)
```

※ **M列＝広告+オーガニック合計**のときはこの式でOKです。合計がL列の場合は `L` に書き換えてください。目標数値シートの合計目標セルが G4 でない場合は `'目標数値'!G4` を実際のセルに変更してください。

**B7**（広告／オーガニック_日割目標・**合計1つ**）※目標数値の日割目標のセルに合わせて変更

```
='目標数値'!J4
```

**B8**（広告／オーガニック_日割実績＝**その日の**合計）

```
=INDIRECT("'オプチャリスト数'!M"&$B$1)
```

※ B2・B4がN1・O1（累計）なので、B2+B4は累計合計になります。日割実績は**その日の**広告+オーガニックなので、データ行のM列（合計列）を参照してください。M列が合計でない場合はその日の合計が入る列に書き換えてください。

**B9**（メルマガ合計・K列）

```
=INDIRECT("'オプチャリスト数'!K"&$B$1)
```

**B10**（特設LINE合計・J列）

```
=INDIRECT("'オプチャリスト数'!J"&$B$1)
```

**B11**（オプチャ合計・E列）

```
=INDIRECT("'オプチャリスト数'!E"&$B$1)
```

---

- **B12・B13**（目標管理シートURL、リスト数シートURL）は **手入力** のままにしてください。
- B6 の `'目標数値'!G4`、B7 の `'目標数値'!J4` は、**広告+オーガニック合計**の目標・日割目標が入っているセルに合わせて変更してください。

### オプチャリスト数の列（再掲）

- B＝日付、D＝オプチャ増加数、E＝オプチャ合計、J＝LINE合計、K＝メール合計、L＝計（場合による）、**M＝広告+オーガニック合計**、N＝オーガニック
- **M列に合計を入れている運用**の場合：目標まで（B6）や「1日あたりの平均」はM列でOK。**広告のみ**の人数・前日比（B2・B3）は、**広告だけの列を別に用意**し、その列を参照するように式の列を書き換えてください。

---

## 6. 使い方の流れ（参照で埋める場合・合計目標用）

1. リスト報告シートの **B1** に「データ行」を入れる（数値の 3, 4, 5… か、上の MATCH の式）。
2. **B2〜B11** に、上記の参照の式を貼る（B6・B7 は目標数値の**合計用**セルに合わせて修正）。
3. **B12・B13** に URL を手入力。
4. **B14** に報告文の式（セクション3）を入れる。
5. 報告文セルをコピーしてチャットやメールに貼って報告。

これで「出てるやつ」をそのまま使え、手動入力は URL と（必要なら）目標数値のセル位置だけになります。
