# 休会5ヶ月経過時リマインド — 何日後？＆送る文

## 5ヶ月後は「何日後」？

**「休会開始日から5ヶ月後の同日」**でリマインドする運用にすると分かりやすいです。

- **日数で言うと**：約 **150日後**（月によって28〜31日あるため、約150〜153日の幅）
- **運用上の目安**：  
  - 休会開始が **〇月1日** → リマインドは **5ヶ月後の1日**（例：2/1開始 → 7/1にリマインド）  
  - 休会開始が **〇月15日** など月途中の場合は、**5ヶ月後の同日前後**に送る

※「〇日後」で固定するより、「休会開始日の5ヶ月後」の**日付**でカレンダーや一覧に登録しておく運用がおすすめです。

## LINE用（短め・そのまま送れる）

```
〇〇さん

お世話になっております。アトリエミー事務局です。

〇〇年〇月よりご休会いただいておりましたが、休会期間がまもなく6ヶ月を迎えます。

規定に基づき、〇〇年〇月頃より**自動的にアトリエミーへ復帰**となります（復帰後の月額は、休会前と同じプランです）。

復帰後は、クレジットカードの定期決済が再開されます。  
カードの変更等ご希望があれば、〇月〇日までにアトリエ生専用LINEまでご連絡ください。

ご不明点があればお気軽にどうぞ。復帰後もよろしくお願いいたします。
```

※ **〇〇さん**・**休会開始月**・**復帰月**・**連絡期限（〇月〇日）** は都度差し替えてください。

## メール用（件名＋本文・事務局名義）

**件名**  
【アトリエミー】1ヶ月後に復帰となります（リマインド）

**本文**

```
〇〇様

お世話になっております。アトリエミー事務局です。

〇〇年〇月よりご休会いただいておりましたが、休会期間がまもなく6ヶ月を迎えます。

規定に基づき、〇〇年〇月〇日頃より**自動的にアトリエミーへ復帰**となります（復帰後の月額は、休会前と同じプランです）。

■ 復帰後のご案内
復帰後は、クレジットカードの定期決済が再開されます。
お支払いカードの変更等ご希望がございましたら、〇月〇日までにアトリエ生専用LINEまでご連絡ください。

ご不明点がございましたら、お気軽にお問い合わせください。
復帰後もどうぞよろしくお願いいたします。

アトリエミー事務局
```

※ **氏名**・**休会開始月**・**復帰月・日**・**連絡期限（〇月〇日）** は都度差し替えてください。
