## 2. 日報チームの課題（お豆さん）

### 2.1 時間の目安（月20時間）

- 25人担当と50人担当で「一緒くた」でよいか不明
- 「月20時間」に**MVP・質問返答・添索**などどこまで含むか曖昧
- 黙々作業会・初心者質問会を主催しない人もいるので、どこまでを範囲にするか
- **新人に「どれくらいかかりますか？」と聞かれたときに示せる表・目安**があると助かる
- リリさん「4時間」は、1人あたり返信5分×週1で妥当かも。週2の人もいるので表があるとよい

### 2.2 報酬設計・ゴールの明確化

- レベル1・2・3の**ゴール**をはっきりさせたい
- **成果報告が出せること＝ゴール**でよいか → **了承**（収益化できている人でないとアトリエに入れないという考え）
- 今月は成果報告していない人も繊維した例あり（りんさん、みっこさん、あこさん等）→ コミュニティ感・同期で仲いい人と行く等の傾向。**成果報告をゴールとする説明は日報メンバーに必要**

### 2.3 レッド部屋・イエローカード（あこさん）

- レッド部屋の**解除基準**を策定済み。特におかしなことは書いていないのでこのまま進める
- **イエローカード**を出す想定ケース:
  - アドバイスに対して感謝や経営が感じられない時
  - 「どうしたらいいですか」ではなく、**質問するときは自分の意見を述べる**
  - 「前も言ったじゃん」のところが改善されていない時
- イエローカードは**3回で再開**。ジャッジは**お豆さん**（お豆の判断で）
- レッド部屋の検査適正テスト: ざっと見ると**2パターン**（運営・お仕事できそうなタイプ / 承認欲求強・ネガティブ強）。**本質黒表が多かった**。レッドカードが出る人も出てきそう

## 3. 報酬・レベル設計の全体方針（ポンさん）

### 3.1 前提

- **CS予算は受講生1人あたり8,300円まで**（変えられない）
- 幹部・事務・日報の先生・添索の先生を除いた**日報・添索メンバー**をレベル1・2・3に分ける
- **レベル1**: 報酬は低めだが、誰でも・駆け出し先生でもやりやすい
- レベルが上がるほど: 1対N、高度な対応、**短時間でしっかり報酬**の構図

### 3.2 体制イメージ

- **おまめさん**: ゴリラ専門にし、**日報担当の幹部をあと1人**引き上げたい
- 部屋が10個以上あるので「部長・課長」的な**幹部補佐（レベル2）**を作りたい
  - 例: 行動が止まった人のケツを叩く、踏み込んだフィードバック、コーチング、**3部屋分を見れる人**
  - 簡単な質問・普段の日報返信はいつもの部屋の先生、MVP選定・いざという時のケツ叩き・ワンオンで引き上げはレベル2

### 3.3 添索のレベル定義（案）

| レベル | 内容 |
|--------|------|
| レベル1 | 基礎課題の添索ができる |
| レベル2 | 自由・自主制作の添索ができる、クライアント数10社以上 |
| レベル3 | クライアント数10社以上＋累計など（要調整） |

- 報酬を「**1人月2,000円**」から「**1課題あたりいくら**」にしたい
- **基礎課題ポートフォリオ完了まで**＝レベル1の先生が担当
- **ポートフォリオ完了後**＝添索者が**レベル2の先生**にバトン（レベル2は基礎課題は見ない、**自主制作・実案件のみ**にコミット）
  - → レベル2の担当人数はかなり増える（例: 4月の自主課題 ナツさん119件、はみさん118件）
- 浮いた報酬はレベル3へ。レベル1は単価は下がるが**件数が増えれば総額はある程度**。レベル2は単価高めで件数も多い

### 3.4 添索者の切り替えタイミング

- **健太さん報酬の歪み**: 途中で萌えさんから切り替わったため初月から高くなった
- **採用案**: **基礎課題ポートフォリオ完了時点で添索者を切り替える** → **採用決定**
- レベル1→レベル2へ変わるタイミングが明確になり、割り振りも見えやすくなる

### 3.5 レベル3（添索）

- マネジメントより**スタープレイヤー**がよい（あゆみさん・にじみさん型）
- マネジメントはナツさん→幹部。**幹部補佐＝もこさん**のように、3部屋分のケツ叩き・トラブルシューティングができる人をレベル3に

- **4月から講師が足りない**。増やしたい。プロモで上振れたら怖い

### 4.4 添索で「大変な人」の傾向（ほらさん）

- **細分化して見れない人**（感覚が正解だと思ってしまう）は講師になると困る。言語化まで時間がかかる
- ルイさん・ファミさん・まヨさんの例:
  - まヨさん: 素直で会話的だが、自分で突き進む。間違ったときはたまに見ないと大変

  - ルイさん: その間。自信がない。**出したくない**（提出のハードルが高い）→ デビューは「もういいや、えいや」で出させる必要
- **つきさん**: 話すとあちこち飛ぶ、文脈が作りづらい。**最後にネクストアクションをまとめて切る**でデビュー。体調管理が苦手でリソースが回りにくい
- 適正検査: つきさんは頭は良いがネガティブ強。ひろみさんは本質ペガサスで「希望を意識」、表面がのんびり。**行かせようと思えばだいたい行ける**が、気持ち・自信の面が大きい

## 6. 日報のレベル設計（難航）

- **レベル1**: 「人生変えるぞ」「とりあえずついてこれ」**ポジティブ**な人
- **レベル2**: **嫌われないかで返信をやめない**。ケツを叩くべきときに叩ける。踏み込んで言える
  - ただし**どこで判断するか（定量化）が難しい**
- **アトリエ繊維率**をレベル2の評価に入れたい（お豆さん案）
  - ともさん（ピンク）は繊維率が低い→「ペイ保証でいい」と言いがち、**お金をかけさせないのが正義**・傷つけない軸でアドバイスしているのでは、という分析

- **定量化は難しい → 主観で見るしかない**という結論に。**千春さんが1回持ち帰り**。収益構造を同じにするため添索とレベル1の収益を下げるので**見直し**する
