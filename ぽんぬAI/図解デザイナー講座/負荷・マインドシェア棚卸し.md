# 負荷・マインドシェア棚卸し（報告用）

今回のプロモで業務をしっかり覚えてもらう意図もあり、**チャレロン終了まで**に別の人に任せたい業務を洗い出すための棚卸しです。  
近藤さん・こあさん・管理官などからの指摘を踏まえ、現状で「マインドシェアをとられているもの」「負荷の高い業務」を書き出し、報告に使います。

---

## 1. マインドシェアをとられているもの

※ 常に頭を占めている・判断に迷う・気になって手が離せない業務やテーマ

| 業務・テーマ | 具体的にどんなことか | 頻度・タイミング |
|--------------|----------------------|------------------|
| （例）配信条件の設計 | 誰に届けるか・除外ラベルの判断が毎回気になる | 配信のたび |
|  |  |  |
|  |  |  |
|  |  |  |

## 2. 負荷の高い業務

※ 時間がかかる・頻度が高い・精神的負担が大きいもの

| 業務名 | 負荷の内容（時間・頻度・負担感） | チャレロン終了まで他に任せられそう？ |
|--------|----------------------------------|--------------------------------------|
| （例）リスト報告（登録経路・特設LINE・オプチャ集計） | 毎日同じ時間にUTAGE・スプシ・オプチャを確認して入力。約〇分 | 手順書があれば代行可能 |
