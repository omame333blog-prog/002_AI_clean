# LINE公式アカウント 垢BAN対策・赤板対策ナレッジ

普段の配信で気をつけているLINEの赤板（垢BAN）対策をナレッジ化したものです。  
※明確に「この単語がNG」と公式に定義されているわけではありませんが、以下を基準に配信時に注意しています。

---

## 1. 公式LINE配信時の「半角スペース」ルール

**公式LINEの配信セット時**は、以下のような文言に**念のため半角スペースを入れて単語として認識されないよう**配信しています。

- 今だけ
- 集客
- 売上
- 絶対
- 必ず
- 無料
- いよいよ
- 人生を変える
- 秘密
- 秘訣
- その他、スパムと見なされやすい表現

**補足：** オプチャ（オープンチャット）では、そのまま配信しています。

---

## 2. 危険なワード・避けるべき表現の基準

### 2-1. 限定的な時間を強調しすぎる表現

緊急感を煽る言葉は、過剰に使用するとスパムと見なされやすいです。  
特に、**繰り返し同じようなメッセージ**を送ると、ユーザーから不信感を抱かれる可能性が高まります。

| 避けたい表現の例 |
|------------------|
| 今だけ |
| 残りわずか |
| 今日が最後のチャンス！ |
| 期間限定！〇〇まで申し込み受付中！ |

**ポイント：** 「今だけ」「期間限定」などを連発しない。本当に期限がある場合のみ、控えめに使う。

---

### 2-2. スパム的な言葉の乱用

セミナー案内などで、**短期間に同じ言葉やフレーズを何度も使う**と「しつこい」と感じられ、スパムとして認識されやすくなります。

| 注意したい言葉の例 |
|-------------------|
| 無料 |
| 今すぐ登録 |
| 絶対 |
| 今すぐ無料登録！ |
| 無料で得られる絶好のチャンス！ |

**ポイント：** 同じ文言の繰り返しを減らし、言い回しを変える。同じ配信内での重複も控える。

---

### 2-3. あいまいで信頼性に欠ける内容

内容があまりにもあいまいで、**具体的な価値やメリットが見えにくいメッセージ**もスパムと見なされがちです。

| 避けたい表現の例 |
|------------------|
| 何か良いことが起こる |
| 驚きの秘密を知りたいですか？ |
| 秘密を知りたいですか？ |
| あなたの人生が変わるかも！ |

**ポイント：** 「何が得られるか」「どんなメリットか」を具体的に伝える。曖昧な煽り文句は使わない。

---

## 3. 配信前チェックのポイント（まとめ）

- [ ] 上記の「危険ワード」に該当する表現はないか
- [ ] 公式LINE配信では、該当語に半角スペースを入れているか
- [ ] 同じ緊急感・煽り表現の繰り返しが多すぎないか
- [ ] 内容が曖昧で信頼性に欠ける書き方になっていないか
- [ ] 無料・絶対・今すぐ等の連発がないか
