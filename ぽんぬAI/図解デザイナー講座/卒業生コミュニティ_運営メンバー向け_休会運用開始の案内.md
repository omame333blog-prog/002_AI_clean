# 運営メンバー向け・休会運用開始の案内（送付用）

運用開始にあたり、運営メンバーに送る案内文です。チャットワークなどにコピペして送ってください。

---

## 送付文（コピペ用）

```
お疲れ様です。

アトリエミー（卒業生コミュニティ）の休会制度の運用を開始します。
運用手順と送付用の文面をまとめたので共有します。

■ フロー（4ステップ）
1. 休会連絡が来る（会員から休会希望の連絡）
2. 決済フォームを送る（休会費 1,000円/月。案内文＋決済リンクをチャットワークで送付）
3. 決済確認後、アトリエ継続課金を停止（該当会員の定期課金を止め、会員に「休会処理完了」をチャットワークで連絡）
4. 連絡ない場合、5ヶ月後にリマインド（休会開始から5ヶ月経過時点で「来月より自動復帰」の案内をチャットワークで送る）

■ 共有資料
・卒業生コミュニティ_休会フロー.md
  → 上記フロー詳細、チャットワークで送る文面（①②③）のコピペ用、サンクスページ文言、定期決済停止の手順など
・卒業生コミュニティ_5ヶ月リマインド文.md
  → 5ヶ月経過時に送るリマインド文（ぽんぬさん名義・メール/LINE用もあり）
・卒業生コミュニティ_休会者リマインド一覧.csv
  → 休会者ごとの「5ヶ月経過日」管理用
・卒業生コミュニティ_UTAGE休会設定の順番.md
  → UTAGEで休会費の商品・決済リンクをセットする順番

■ 担当まわり
・決済フォーム送付・休会処理完了の連絡・リマインド送付：事務局（または担当者）
・アトリエ継続課金の停止操作：こはるさん・まるりさん・あすぽんさん（役割はフロー内のチェックリスト参照）

不明点や運用で揃えたいことがあれば、チャットワークで共有してください。
よろしくお願いします。
```

---

※ 共有資料は、実際の保管場所（Googleドライブ・Notion・共有フォルダのパスなど）に合わせて「・」以下を差し替えてください。
