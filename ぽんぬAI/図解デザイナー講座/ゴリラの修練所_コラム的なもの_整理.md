# ゴリラの修練所・コラム的なもの 整理メモ

---

## 用語の整理（2種類ある）

### 1. ゴリラの修練所（模擬案件OJT）

- **別名**：ゴリラの修練場
- **内容**：**模擬案件**をチームでこなす場所。実案件さながらの速度感・クオリティでスライド作成
- **開始**：2025年10月
- **特徴**
  - ディレクター育成が間に合わず受託を一部お断りしていた背景から開設
  - 依頼主＝ぽんぬとして、ポートフォリオに「株式会社ミコマで作成した模擬案件」として掲載可能
  - ディレクターのOJT場所でもあり、受講生のOJT場所でもある
- **対象**：ゴールドランク以上、課題クリア、コミュニケーション取れる人
- **出典**：ぽんぬのつぶやき 2025/10/19「ゴリラの修練所カミングスーン」

---

### 2. ゴリラのつぶやき（講師からのコラム的発信）

- **幹部MTGでの議論**：2025年12月6日
- **背景**：ポンヌのつぶやき（音声）以外に、**講師からも届けたい**ものがあった
- **狙い**
  - ポンヌさんが忙しい時に、**全体運営から届けたいこと**を発信したい
  - 「神々のお告げ」的な部屋にする案 → 結局「**ゴリラのつぶやき**」に
  - ポンヌ・お豆・コラから届けられる形

#### 構成・運用イメージ

| 項目 | 内容 |
|------|------|
| 発信者 | ぽんぬ・お豆さん・コラさん |
| 形式 | **テキストでOK**（ラジオ/音声である必要はない） |
| 頻度 | お豆さんとコラさんが**交互にたまに**つぶやく |
| ぽんぬの位置 | 別で「ぽんぬの部屋」が欲しい。ゴリラのつぶやきは講師側の声 |
| 配置 | インフォメーションに「ゴリラのつぶやき」を入れる |

#### コラさん・お豆さん講師のつぶやき

- **コラさんのコラム**のようなもの（講師からのラム、コラム）
- ポンヌさんが忙しい時・間に合わない時に、**一声かけたい時**用
- つぶやきというより**コラム的**なテキスト
- **著作権**：コラさんがテキストを作成 → ぽんぬに見せて添削受けてから出す
- 長さ：つぶやき程度でOK、長くなくてよい

#### 決定事項（2025年12月MTG）

- ゴリラのつぶやき = 講師（コラさん・お豆さん）のつぶやき
- インフォメーションと「お仕事」を分ける。インフォメーション側にゴリラのつぶやき
- 勉強会の部屋とは別
- コラさんに著作権まわりのテキスト作成を依頼

## 3月MTG（2026/03/03）でのゴリラのつぶやき

- **状況**：止まっている
- **方針**：再開する
- **テーマ**：「**人の力を借りよう**」で前に進もう → その後に「添削に出すのは怖くない」
- **表現**：「頼る」ではなく「**借りる**」。「人の力を借りる」が合っている
- **事例**：1月のお部屋で「スライドを作り始めて2週間経ちました。今でも出せません」→ 共感の返信が多く、「傷の舐め合い」から癒されて前に進めるフェーズがあることを再認識
- **メッセージ**：自立＝1人で頑張る人ではなく、**頼る先が数えきれない人**。人の力を借りられることが大切

## まとめ

| 用語 | 中身 |
|------|------|
| **ゴリラの修練所** | 模擬案件OJT。チームでスライドを作る練習場 |
| **ゴリラのつぶやき** | 講師（ぽんぬ・お豆・コラ）のコラム的なテキスト発信。ポンヌ以外から運営の声を届ける |
| **コラム的なもの** | コラさん・お豆さんが書く、講師からのテキスト。著作権はコラさんが整理 |

「ゴリラの修練所でコラム的なものを書く」という表現は、  
**ゴリラのつぶやき**（講師のコラム的発信）のことを指していると考えられる。
