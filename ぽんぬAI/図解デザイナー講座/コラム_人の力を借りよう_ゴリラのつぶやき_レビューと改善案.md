# コラム「人の力を借りよう」全体レビュー

---

## 現在のコラム案

「人に迷惑をかけたくない」——そう思う方は多いと思います。でも、せっかく講座に入ったからこそ、「できない」のは当たり前です。

初めてデザインを学ぶ方もいれば、他のスクールで経験を積んでからの方もいます。日報や添削に挑戦中の方も、それぞれスタート地点は違います。それでも、みなさんがこの講座に入った理由は「誰かと比べるため」ではなく、**自分を変えたいから**ではないでしょうか。

だからこそ、まず一歩として、こんなことを誰かに話してみてください。

- ちょっと難しいなと思っていること
- 不安に感じていること
- 「どうしよう」と迷っていること

日報に書いてもいいですし、個別のDMでも大丈夫です。「何が不安なんだろう」と書くだけで、必ず誰かがそれを見ています。

一方で、心の中に閉じておくだけだと、SNSにも出てきません。周りからは「この人は問題なさそうだな」としかわかりません。みなさんは大人なので、一人で抱え込むこともできます。でも、そのために、前に進むきっかけを逃してしまっていませんか？

この講座は、マンツーマンコンサルではなく**コミュニティ**です。ぽんぬさんをはじめ運営メンバーがいるのはもちろん、同じ時期にスタートした同期の部屋や、雑談部屋で先輩と話す機会もたくさんあります。

ネガティブなことでも、ぜひ共有してみてください。誰かの力を借りる一歩は、そこから始まります。

## 全体レビュー

### 良い点

| 項目 | 内容 |
|------|------|
| 共感の入口 | 「迷惑かけたくない」でネガティブ属性の人に寄り添えている |
| 前提の整理 | 「できないのは当たり前」「自分を変えたいから入った」で比較軸から外している |
| 具体例 | 話す内容（難しい・不安・どうしよう）が具体的 |
| 出口の明示 | 日報・DM、同期の部屋・雑談部屋など、借りる相手を提示している |
| 締め | 「ネガティブでも共有を」で行動を促している |

### LPナレッジ形式とのギャップ

| 項目 | 危機感版コラム | 今回のコラム | 改善案 |
|------|----------------|--------------|--------|
| タイトル案 | 複数パターンあり | なし | 複数案を追加 |
| セクション見出し | ■で区切り、スキャンしやすい | なし | ■見出しを入れる |
| 段落 | 1〜2行で改行多め | やや長め | 短く区切る |
| 対比 | 「〜ではなく〜」の構造が明確 | やや淡白 | 心の中だけ vs 書く、の対比を強調 |
| 次の一歩 | 最後に行動を一文で | やや抽象 | 「今日の日報に1行でも」など、もう少し具体化 |

### 内容面の確認

- **3・4・5番**（頼れない理由・心の中だけでは届かない・誰に借りるか）は網羅できている ✓
- **添削・次の行動**は次の担当者向けなので含めていない ✓
- **「借りる」という表現**は3月MTGで推奨されているが、本文には「人の力を借りる」が1回のみ。もう少し意識して入れる余地あり
