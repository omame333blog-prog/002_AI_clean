# 女性創業者セミナー＆交流会 — フォローアップ挨拶の叩き台

知り合った方によって内容を変えられるよう、パターンをいくつか用意しました。名刺交換した方・特に話した方・興味が近い方など、相手に合わせて選んでください。

---

## 【シンプル版】汎用

```
〇〇様

本日は女性創業者セミナー＆交流会でお話しできてありがとうございました。
短い時間でしたが、〇〇（例：お仕事の内容／〇〇のご経験など）をお伺いでき、大変勉強になりました。

また機会がございましたら、ぜひお話しできれば幸いです。
今後ともどうぞよろしくお願いいたします。
```

---

## 【少しカスタム入り】話した内容に触れる場合

```
〇〇様

本日は横浜市信用保証協会のセミナー＆交流会でお会いできてありがとうございました。
〇〇（例：〇〇の事業内容／創業期のご苦労話／今後のご予定など）についてお話を伺い、とても参考になりました。

またお話できる機会がありましたら嬉しいです。
どうぞよろしくお願いいたします。
```

---

## 【名刺交換メイン】深く話せなかった方

```
〇〇様

本日は交流会で名刺を交換させていただき、ありがとうございました。
お忙しい中、ほんの少しでしたがお話しできて嬉しかったです。

またセミナーや交流会などでお会いできましたら、ぜひお話を伺えたらと思います。
今後ともどうぞよろしくお願いいたします。
```

---

## 【つながりを大切に】今後も交流したい場合

```
〇〇様

本日は女性創業者セミナー＆交流会でお話しできてありがとうございました。
〇〇（例：同じ創業期ならではの悩み／横浜で活動されていることなど）を共有でき、励みになりました。

またこうした場や、機会があればお茶などでもお話しできれば嬉しいです。
今後ともどうぞよろしくお願いいたします。
```

---

## 使い方のポイント

- **〇〇** の部分に、相手の名前・事業・話した内容の一言などを入れてください。
- LINE・メール・SNSなど送る媒体に合わせて、改行や「です・ます」のトーンはそのまま使っても、少し崩しても大丈夫です。
- 送るタイミング：**翌日～2〜3日以内**が、覚えていてもらいやすくおすすめです。

必要に応じて文面を短くしたり、自分らしい言い回しに変えて使ってください。
