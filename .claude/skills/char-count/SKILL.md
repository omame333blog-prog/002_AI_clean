# 文字数カウント

ユーザーが提供したテキストの文字数を、以下の4パターンで計測して表示する。

## 4パターンの定義

| パターン | 内容 |
|---|---|
| 改行含む・スペース含む | すべての文字をそのままカウント |
| 改行含む・スペース除く | 半角スペース・全角スペース・タブを除いてカウント |
| 改行除く・スペース含む | 改行文字（\n, \r）を除いてカウント |
| 改行除く・スペース除く | 改行・スペース類をすべて除いてカウント |

スペース類の定義：半角スペース（` `）、全角スペース（`　`）、タブ（`\t`）

## 手順

1. ユーザーからテキストを受け取る（メッセージ内に含まれているか、ファイル指定か確認する）
2. `scripts/count_chars.py` を使って4パターンをカウントする：
   ```bash
   echo "テキスト" | python3 scripts/count_chars.py
   # またはファイルの場合
   cat ファイルパス | python3 scripts/count_chars.py
   ```
3. 結果を以下の形式で見やすく出力する：

```
【文字数カウント結果】

| パターン              | 文字数 |
|---------------------|-----:|
| 改行含む・スペース含む  | XXX  |
| 改行含む・スペース除く  | XXX  |
| 改行除く・スペース含む  | XXX  |
| 改行除く・スペース除く  | XXX  |
```

## 注意

- テキストがメッセージ内にある場合はそのまま処理する
- ファイルパスが指定された場合はファイルを読み込んで処理する
- テキストが長い場合も省略せず正確にカウントする
- スクリプトのパスはこのSKILL.mdからの相対パス（`scripts/count_chars.py`）
