日報担当者の活動状況（返信・MVP発表）を確認します。

## 実行コマンド

```bash
cd /Users/kasaimami/002_AI_ && source venv/bin/activate && set -a && source .discord_token && set +a && python check_tantosha_activity.py $ARGUMENTS
```

引数：
- 引数なし → 直近4週分を画面表示
- `--weeks 8` → 直近8週分
- `--month 2026-03` → 指定月の全週
- `--post` → 集計結果をDiscord（#🤖おまめ使用中）に投稿

## 判定ルール

- **返信 ✅**: 担当者のメッセージが期間内に1件以上ある（件数は表示しない）
- **返信の除外**: `<@&ロールID>` を含む全員宛メッセージ（交流会告知・MVP発表・全体連絡）はカウントしない
- **MVP ✅**: 誰かのメッセージに「MVPおめでとう」または担当者の「MVP」+「発表/👑」がある

## 出力形式

```
🩷日報：ピンク（担当：ともも_運営）
  03/16〜03/22  返信 ✅  MVP発表 ✅
```

件数（例：`（37件）`）は表示しない。✅/❌のみ。

## --post モードの集計期間

金曜実行を想定（例: 03/20実行）:
- **返信期間**: 実行日から前7日間（03/13〜03/19）
- **MVP期間**: 当週月曜〜前日（03/16〜03/19）

> ⚠️ 通常表示（引数なし）はカレンダー週（月〜日）で判定するため、日曜の返信が翌週の❌に見える場合がある。
> `--post` は直近7日間なので週またぎの返信も正しく✅になる。表示上の❌は誤りではない。
