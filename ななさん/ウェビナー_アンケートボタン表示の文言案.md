# ウェビナー「一定時間経過でアンケートボタン表示」の文言案

## 前提
- **常時表示**：※動画を最後まで見てアンケート回答するとプレゼントがあります🎁／※動画を最後まで見ないとアンケートは送られてきません
- **一定時間経過後**：アンケート申込ボタンを表示する

---

## 1. 常時表示に足す「説明」（ボタンが出る理由）

視聴者が「今はボタンがない」と混乱しないように、最初から次のどれか1行があるとよいです。

| 案 | 文言 |
|----|------|
| A | 視聴が一定時間経過すると、下にアンケート申込ボタンが表示されます。 |
| B | 動画を一定時間ご視聴いただくと、アンケート申込ボタンが表示されます。 |
| C | しばらくご視聴いただくと、アンケートのお申し込みボタンが表示されます。 |

※「一定時間」を具体的に出すか（例：約○分経過後）は、仕様や視聴時間に合わせて調整。

---

## 2. ボタンが出たときの「見出し・文言」

ボタンと一緒に表示する短い文です。

| 案 | 文言 | トーン |
|----|------|--------|
| 1 | ここまでご視聴いただきありがとうございます！<br>下のボタンからアンケートにご回答ください。 | 感謝＋依頼 |
| 2 | 動画の所要時間が経過しました。<br>アンケートにご協力ください。下のボタンをタップしてください。 | 仕組みの説明＋依頼 |
| 3 | アンケート回答の受付を開始しました。<br>プレゼント受け取りは、下のボタンからお申し込みください。 | 受付開始＋プレゼント訴求 |
| 4 | ご視聴ありがとうございます。<br>動画を最後まで見た方は、下のボタンをタップしてアンケートにご回答ください。 | 元の文言に近い・自然 |

---

## 3. 組み合わせ例（常時＋ボタン表示時）

**常時表示ブロック**
```
※動画を最後まで見てアンケート回答するとプレゼントがあります🎁
※動画を最後まで見ないとアンケートは送られてきません
視聴が一定時間経過すると、下にアンケート申込ボタンが表示されます。
```

**一定時間経過後に表示するブロック**
```
ここまでご視聴いただきありがとうございます！
下のボタンをタップして、アンケートにご回答ください。

[アンケートに申し込む]  ← ボタン
```

---

## 4. 注意したい点
- 「最後まで見ないとアンケートは送られてこない」と「一定時間でボタン表示」は、仕組みとしては「一定時間＝最後まで見たとみなす」なので、**「一定時間ご視聴いただくとボタンが表示されます」**のように書いておくと矛盾しにくい。
- プレゼント目的で早送りする人対策が必要なら、「○分以上視聴」などの条件を、常時表示の説明に軽く入れておくのもあり。
