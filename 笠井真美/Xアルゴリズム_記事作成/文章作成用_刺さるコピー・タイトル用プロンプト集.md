# 文章作成用｜刺さるコピー・タイトル用プロンプト集

記事・ブログ・SNSの**タイトル・キャッチコピー**をAIで作るときに使うプロンプトです。  
「〇〇」にテーマを入れて、そのままChatGPTやClaudeに貼って使えます。

---

## 使い方

1. **テーマを決める** … 記事の内容・商品・サービスなど
2. **使いたいテクニックを選ぶ** … 下の一覧から1つ（または組み合わせ）
3. **プロンプトの「〇〇」をテーマに置き換えてコピペ**
4. **出力を記事のタイトル・見出し・リード文に使う**

※複数テクニックを組み合わせたいときは、プロンプトの末尾に「また、〇〇（別テクニック）の要素も入れてください」と追記するとよいです。

---

## プロンプト一覧（コピペ用）

### ① 禁止命令で好奇心を刺激（カリギュラ効果）

```
「〇〇」についてのキャッチコピーを作ってください。ただし、「〇〇するな」「〇〇しないでください」「〇〇禁止」のような禁止命令の形にしてください。人は禁止されると逆にやりたくなる心理があるので、それを利用したコピーにしてください。10個お願いします。
```

---

### ② 数字で具体性を出す

```
「〇〇」についてのキャッチコピーを10個作ってください。必ず具体的な数字を入れてください。例：「3日で」「97%が知らない」「1万円節約」など。数字はできるだけ意外性のあるものにしてください。
```

---

### ③ 問いかけで自分事化させる

```
「〇〇」についてのキャッチコピーを10個作ってください。すべて「疑問形」にしてください。読み手が「ドキッ」とするような、自分のことを言われているような問いかけにしてください。
```

---

### ④ 意外性で目を止めさせる（常識の逆）

```
「〇〇」についてのキャッチコピーを10個作ってください。一般的な常識とは「逆」のことを主張するコピーにしてください。「普通はAだと思われているけど、実はBだ」という構造にしてください。
```

---

### ⑤ 感情ワードで心を揺さぶる

```

```

```

---

```

---

```

---

```

```

```

## 組み合わせ例

- **数字 ＋ 禁止** … 「この〇〇、3日間だけ使うのやめて」（具体性＋好奇心）
- **問いかけ ＋ ターゲット** … 「〇〇で悩んでる人、そのやり方続けてますか？」
- **ストーリー ＋ 失敗談** … 「〇〇で失敗した私が、今は△△できている理由」
- **意外性 ＋ データ風** … 「調査で判明。〇〇は実は逆効果だった」

必要に応じて、プロンプトの最後に「上に加えて、〇〇の要素も入れてください」と書くと、複数テクニックを一度に試せます。
